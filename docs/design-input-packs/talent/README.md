# Talent — design input packs

**Code:** `packages/talent` (Development Plan / IDP, career journeys, talent admin, etc.).

**Exports:** Save DIPs and related Markdown under [`exports/`](./exports/).

Prefer **one DIP per slice** (e.g. employee IDP submit vs manager approve vs admin cycle config) rather than one file for the entire Talent module.

When starting a Cursor session, **@ mention** the active DIP under `exports/`.
